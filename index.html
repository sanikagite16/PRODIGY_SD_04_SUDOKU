📖 SudokuVerse Game Guide

🎯 Welcome to SudokuVerse!

SudokuVerse is an interactive Sudoku game designed to provide a fun, challenging and rewarding experience for players of all skill levels. Every puzzle you complete earns XP, helps you level up, unlocks achievements and improves your personal statistics.

This guide explains every feature, scoring rule and gameplay mechanic used in SudokuVerse.

🎯 Objective

Fill every empty cell so that:

  • Each row contains the numbers 1–9 exactly once.
  • Each column contains the numbers 1–9 exactly once.
  • Each 3×3 box contains the numbers 1–9 exactly once.
  • Press Finish to verify your completed puzzle.
  • Incorrect puzzles will not be accepted.

🎮 Game Modes

Mode Description
▶ Play Game Choose any difficulty and solve Sudoku puzzles.
🔥 Daily Challenge Complete today's special puzzle to earn bonus XP.
🤖 AI Solver Enter any valid Sudoku puzzle and let the AI solve it instantly.

📊 Difficulty & XP Rewards

Every completed Sudoku puzzle rewards XP based on its difficulty.

Difficulty Base XP Description
🟢 Easy 25 XP Best for beginners and quick practice.
🟡 Medium 50 XP Balanced challenge.
🟠 Hard 75 XP Requires stronger solving skills.
🔴 Expert 100 XP Designed for experienced players.

Note: This is your Base XP. Hint penalties and Daily Challenge bonuses are applied afterwards.

🔥 Daily Challenge

A unique Daily Challenge is available every day. Completing it rewards you with extra bonus XP in addition to your puzzle XP.

Day Difficulty Bonus XP
Sunday 🟢 Easy +25 XP
Monday 🟡 Medium +50 XP
Tuesday 🟠 Hard +75 XP
Wednesday 🔴 Expert +100 XP
Thursday 🟡 Medium +50 XP
Friday 🟠 Hard +75 XP
Saturday 🟢 Easy +25 XP

Bonus XP is added after your puzzle XP has been calculated.

💡 Hint System

Hints help you solve difficult puzzles, but using them reduces the XP earned from the puzzle.

Hints Used Puzzle XP Awarded
0 100% of Base XP
1 90% of Base XP
2 80% of Base XP
3 70% of Base XP
4 or more 60% of Base XP (Minimum)

Example

Easy Puzzle = 25 XP

Example: The player used 5 hints.

Puzzle XP = 15 XP

Puzzle XP never drops below 60% of the Base XP, even if more than 4 hints are used.

If it is today's Daily Challenge:

15 XP + 25 Bonus XP = 40 XP Total

⚠ Important Rules

  • ✔ XP is awarded only after successfully completing a Sudoku puzzle.
  • ✔ Incorrect Sudoku solutions do not receive XP.
  • ✔ Daily Challenge Bonus XP is added after Puzzle XP is calculated.
  • ✔ Using hints reduces Puzzle XP.
  • ✔ Puzzle XP never drops below 60% of the Base XP.
  • ✔ Level Up occurs automatically when your Total XP reaches the next level.
  • ✔ Statistics update automatically after every completed game.
  • ✔ AI Solver does not award XP, achievements or increase your streak.
  • ✔ Daily Challenge difficulty changes automatically every day.
  • ✔ All game progress is saved automatically.
