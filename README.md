# 🧩 SudokuVerse AI

A responsive Sudoku Game and AI-powered Sudoku Solver built using **HTML, CSS, and JavaScript** as **Task-04** during my **Software Development Internship at Prodigy InfoTech**.

## 🚀 Features
- Play Sudoku with multiple difficulty levels
- AI Sudoku Solver using the Backtracking Algorithm
- Timer, hints, score tracking & sound effects
- Multiple themes
- Fully responsive design for Desktop, Tablet & Mobile

## 🛠 Tech Stack
HTML5 • CSS3 • JavaScript (ES6) • Backtracking Algorithm • DOM Manipulation • Local Storage • Git • GitHub • Vercel
